Features

不是一个聊天机器人，是一个工作流

01三段式尽调报告

好生意 / 好公司 / 好价格，每段强制 PASS / FAIL / GRAY，不允许"也许可以考虑"的模糊结论。

02多 Agent 圆桌辩论

巴菲特 vs 段永平 vs 芒格，观点冲突必须显式呈现。你读到的是争论，不是共识灌输。

03能力圈档案

本地 SQLite 记录你"真懂的生意"。每次分析先问你"能 30 字讲清它吗"，答不出就拒绝分析。

04持仓周报

自动读取你的持仓，每周生成三段式复盘。可推送到企微 / Telegram / Obsidian Vault。

05反幻觉防线

Decimal 精度财务计算 · Benford's Law 异常检测 · 多源交叉验证 · 强制 tool call 数据来源。

06本地优先

所有数据留本地，持仓数据从不离开你的电脑。支持 Ollama 纯本地运行，无 API 焦虑。
